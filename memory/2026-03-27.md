## 📚 学习收获

1. **Dify 部署技巧**：
   - 使用 `--profile postgresql` 启动 PostgreSQL 支持
   - 使用 `--profile chroma` 启动 ChromaDB 向量数据库
   - 修改 `VECTOR_STORE=chroma` 配置使用 ChromaDB

2. **Ollama 模型管理**：
   - 使用 `ollama pull` 下载模型
   - 使用 `ollama list` 查看已安装模型
   - 使用 `curl` 测试 API 接口

3. **Docker 容器管理**：
   - 使用 `docker compose` 管理容器
   - 使用 `docker exec` 进入容器执行命令
   - 使用 `docker logs` 查看容器日志
