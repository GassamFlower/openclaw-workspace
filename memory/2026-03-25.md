## 🐛 遇到的问题

### 问题1：飞书机器人不回复
- **问题描述**：配置完成后，机器人不回复消息
- **解决方法**：
  1. 检查 Gateway 日志
  2. 发现缺少 `im:message:send` 权限
  3. 在飞书开放平台添加该权限
  4. 重启 Gateway
- **经验教训**：配置工具前先查看文档和日志，不要盲目尝试

### 问题2：Cron 任务参数错误
- **问题描述**：首次配置 cron 时，`payload.kind` 和 `sessionTarget` 参数不对
- **解决方法**：
  1. 查看文档，发现需要用 `agentTurn` 类型
  2. 使用 `sessionTarget: "current"` 绑定当前会话
  3. 重新配置任务
- **经验教训**：工具的参数配置要严格按照文档来，不要自己瞎猜
