# 每日边做边学提醒任务

**任务名称**：边做边学提醒
**触发时间**：每天 10:00
**执行频率**：每天
**执行方式**：agentTurn

---

## 任务描述

每天10点提醒自己：
1. 今天学到了什么？
2. 遇到的问题和解决方案
3. 明天的学习计划

## 执行内容

```json
{
  "message": {
    "content": "⏰ 10:00 边做边学提醒\n\n【今日学习回顾】\n\n1. 今天学到了什么？\n   - 技术点1：...\n   - 技术点2：...\n\n2. 遇到的问题\n   - 问题1：解决方案\n   - 问题2：解决方案\n\n3. 明天改进\n   - 改进点1\n   - 改进点2\n\n【学习资源】\n- 学习方法论：学习方法论.md\n- 快速执行：快速执行指南.md\n\n💡 记住：能记录的不要记在脑子里！",
    "attachments": []
  }
}
```

---

## 备注

- 每天开发后花10分钟填写学习回顾
- 记录在 memory/YYYY-MM-DD.md
- 定期回顾学习方法论
