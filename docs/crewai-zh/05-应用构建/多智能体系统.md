# 多智能体系统 - 复杂系统实战

> 构建复杂的多智能体协作系统

## 🎯 应用场景

- 内容创作团队
- 产品开发团队
- 客服团队
- 研发团队

---

## 💻 基础示例

```python
from crewai import Agent, Task, Crew

# 创建多个 Agents
researcher = Agent(role="研究员", goal="收集信息")
writer = Agent(role="写作者", goal="撰写文章")
editor = Agent(role="编辑", goal="编辑内容")

# 创建 Tasks
research_task = Task(description="收集信息", agent=researcher)
writing_task = Task(description="撰写文章", agent=writer)
editing_task = Task(description="编辑内容", agent=editor)

# 创建 Crew
crew = Crew(
    agents=[researcher, writer, editor],
    tasks=[research_task, writing_task, editing_task]
)

# 运行
result = crew.kickoff()
```
