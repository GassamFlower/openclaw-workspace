# 工作流编排 - 工作流

> 编排 Agent 工作流

## 🎯 流程类型

- 顺序执行: task1 → task2 → task3
- 层级执行: manager → worker
- 混合执行: 根据任务类型选择

---

## 💻 使用示例

```python
# 顺序执行
crew = Crew(
    agents=[agent1, agent2],
    tasks=[task1, task2],
    process="sequential"
)

# 层级执行
crew = Crew(
    agents=[manager, worker],
    tasks=[task1, task2],
    process="hierarchical"
)
```
