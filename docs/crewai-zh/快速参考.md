# CrewAI 快速参考

## 📚 核心概念

### Agent
AI 专家,有自己的角色、目标和技能

### Task
工作描述,定义 Agent 要做什么

### Crew
团队管理器,协调多个 Agent

### Tool
Agent 可以调用的外部功能

### Process
执行方式: 顺序/层级/混合

---

## 💻 基础使用

```python
from crewai import Agent, Task, Crew

# 创建 Agent
researcher = Agent(
    role="研究员",
    goal="收集信息",
)

# 创建 Task
task = Task(
    description="收集关于 {topic} 的信息",
    agent=researcher
)

# 创建 Crew
crew = Crew(
    agents=[researcher],
    tasks=[task],
)

# 运行
result = crew.kickoff(inputs={"topic": "AI"})
``````

---

## 🎯 常用属性

### Agent 属性
- `role`: 角色
- `goal`: 目标
- `backstory`: 背景故事
- `tools`: 工具列表
- `memory`: 是否启用记忆

### Task 属性
- `description`: 任务描述
- `agent`: 负责的 Agent
- `expected_output`: 预期输出

### Crew 属性
- `agents`: Agent 列表
- `tasks`: Task 列表
- `process`: 执行方式
- `verbose`: 是否显示详情

---

## 💡 设计技巧

1. **清晰的描述**: 明确的任务描述
2. **合理的工具**: 3-5 个工具
3. **明确的预期输出**: 说明期望的结果
4. **适当的委派**: Manager 可以委派,Worker 不能
