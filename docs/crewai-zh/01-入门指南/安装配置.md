# CrewAI 安装配置

> 从零开始搭建 CrewAI 开发环境

## 🛠️ 环境准备

### 1. 安装 Python

CrewAI 需要 Python 3.8+:

```bash
# 检查 Python 版本
python --version
# 推荐: 3.10 或更高版本

# 如果没有安装,从官网下载: https://www.python.org/downloads/
```

### 2. 安装 uv (推荐)

CrewAI 推荐使用 `uv` 作为包管理器:

```bash
# Windows (PowerShell)
irm https://astral.sh/uv/install.ps1 | iex

# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## 📦 安装 CrewAI

### 方式一: 使用 uv (推荐)

```bash
# 创建项目
uv init crewai-tutorial
cd crewai-tutorial

# 安装 CrewAI
uv add crewai
uv add --dev crewai[dev]

# 安装常用工具
uv add crewai-tools
```

### 方式二: 使用 pip

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 升级 pip
pip install --upgrade pip

# 安装 CrewAI
pip install crewai
pip install crewai-tools
```

## ✅ 验证安装

创建测试脚本 `test_install.py`:

```python
from crewai import Agent, Task, Crew

# 创建简单 Agent
researcher = Agent(
    role="研究员",
    goal="研究技术",
    backstory="你是一位专家",
    verbose=True
)

# 创建任务
task = Task(
    description="总结技术趋势",
    agent=researcher
)

# 创建 Crew
crew = Crew(agents=[researcher], tasks=[task], verbose=True)

# 运行
try:
    result = crew.kickoff()
    print("✅ 安装成功!")
    print(result)
except Exception as e:
    print(f"❌ 安装失败: {e}")
```

运行测试:

```bash
# Windows
python test_install.py

# macOS/Linux
python3 test_install.py
```

## 🎯 快速体验

### 最简单的 Crew

```python
from crewai import Agent, Task, Crew

# 创建 Agent
agent = Agent(
    role="助手",
    goal="帮助用户",
    backstory="你是一个有用的 AI 助手"
)

# 创建任务
task = Task(
    description="用一句话介绍你自己",
    agent=agent
)

# 创建并运行 Crew
crew = Crew(agents=[agent], tasks=[task], verbose=True)
result = crew.kickoff()
```

### 使用工具

```python
from crewai import Agent, Task, Crew
from crewai_tools import SerperDevTool

# 创建工具
search_tool = SerperDevTool()

# 创建 Agent
agent = Agent(
    role="研究员",
    goal="研究信息",
    tools=[search_tool],
    verbose=True
)

# 创建任务
task = Task(
    description="搜索 AI 最新进展",
    agent=agent
)

# 运行
crew = Crew(agents=[agent], tasks=[task], verbose=True)
result = crew.kickoff()
```

## 🐛 常见问题

### Q1: pip 安装失败

```bash
# 升级 pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install crewai -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: API 密钥无效

- 检查密钥是否正确
- 确认 API 有余额
- 设置环境变量

### Q3: 模块找不到

```bash
# 检查安装
pip list | grep crewai

# 重新安装
pip install --force-reinstall crewai
```

### Q4: uv 和 pip 混用问题

- 建议只使用一个包管理器
- 如果用 uv,不要同时用 pip
- 清理缓存: `uv cache clean`
