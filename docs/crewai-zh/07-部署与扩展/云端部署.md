## 💻 Flask 部署

```python
from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route('/chat', methods=['POST'])
def chat():
    data = request.json
    response = agent.invoke(data['question'])
    return jsonify({"response": response.content})

if __name__ == '__main__':
    app.run(port=5000)
```
