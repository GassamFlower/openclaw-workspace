# 常见问题 - FAQ

## 📖 CrewAI 常见问题

### Q: CrewAI 和 LangChain 有什么区别?
A: CrewAI 专注于多智能体协作,LangChain 更通用。

### Q: 如何选择 Process 类型?
A: 简单任务用 Sequential,复杂任务用 Hierarchical。

### Q: 如何优化 Agent 性能?
A: 使用缓存、减少工具数量、优化 Prompt。

### Q: 如何实现记忆功能?
A: 在 Agent 配置中启用 memory 参数。
