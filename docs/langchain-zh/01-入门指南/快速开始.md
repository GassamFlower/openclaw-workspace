# LangChain 快速开始

> 5 分钟内跑通你的第一个 LangChain 应用

## 🚀 第一步: Hello World

### 最简单的对话

```python
from langchain_openai import ChatOpenAI

# 创建模型
llm = ChatOpenAI(model="gpt-3.5-turbo")

# 对话
response = llm.invoke("用一句话介绍你自己")
print(response.content)
```

**预期输出**:
```
我是一个由OpenAI开发的大型语言模型,旨在帮助人们回答问题、提供信息和完成任务。
```

## 🎯 第二步: 创建第一个 Agent

### 什么是 Agent?

Agent = 会自主决策的 AI 助手

它知道:
- 需要什么工具
- 什么时候调用工具
- 怎么综合信息回答

```python
from langchain.agents import create_agent

# 创建 Agent
agent = create_agent(
    model="gpt-3.5-turbo",
    tools=[],
    system_prompt="你是一个有用的助手",
)

# 运行
result = agent.invoke({
    "messages": [{"role": "user", "content": "请帮我写一段 Python 快速排序算法"}]
})

print(result)
```

## 🔥 第三步: 使用工具

### 给 Agent 赋予能力

```python
from langchain.agents import create_agent
from langchain.tools import tool

# 定义工具
@tool
def calculate(expression: str) -> str:
    """计算数学表达式,例如: 1 + 2 * 3"""
    try:
        result = eval(expression)
        return f"计算结果: {result}"
    except Exception as e:
        return f"计算错误: {e}"

@tool
def get_current_time() -> str:
    """获取当前时间"""
    from datetime import datetime
    return f"当前时间: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}"

# 创建 Agent
agent = create_agent(
    model="gpt-3.5-turbo",
    tools=[calculate, get_current_time],
    system_prompt="你是一个计算助手",
)

# 测试
print(agent.invoke({
    "messages": [{"role": "user", "content": "1+2*3 等于多少? 当前时间?"}]
}))
```

**Agent 会自动**: 先调用 `calculate`,再调用 `get_current_time`

## 📊 第四步: 使用记忆

### 多轮对话

```python
from langchain.agents import create_agent
from langchain.memory import ConversationBufferMemory

# 创建记忆
memory = ConversationBufferMemory()

# 创建 Agent
agent = create_agent(
    model="gpt-3.5-turbo",
    tools=[],
    system_prompt="你是一个有用的助手",
)

# 第1轮对话
response1 = agent.invoke({
    "messages": [{"role": "user", "content": "我叫小明"}]
}, memory=memory)
print(response1)

# 第2轮对话 - Agent 记得你叫小明
response2 = agent.invoke({
    "messages": [{"role": "user", "content": "你好"}]
}, memory=memory)
print(response2)
```

## 🗂️ 第五步: 基于知识库问答

### RAG (检索增强生成)

```python
from langchain.agents import create_agent
from langchain_community.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings

# 准备知识库
knowledge_base = [
    "LangChain 是一个用于开发 LLM 应用的框架",
    "LangChain 支持多种模型提供商",
    "LangChain 提供工具调用和记忆功能",
]

# 创建向量数据库
vectorstore = FAISS.from_texts(
    texts=knowledge_base,
    embedding=OpenAIEmbeddings(),
)

# 创建检索器
retriever = vectorstore.as_retriever()

# 创建 Agent
agent = create_agent(
    model="gpt-3.5-turbo",
    tools=[retriever],
    system_prompt="你是一个知识助手",
)

# 问答
print(agent.invoke({
    "messages": [{"role": "user", "content": "LangChain 支持哪些模型?"}]
}))
```

## 📝 完整示例: 智能客服 Agent

### 组合多种功能

```python
from langchain.agents import create_agent
from langchain.tools import tool
from langchain.memory import ConversationBufferMemory

# 工具 1: 查询订单
@tool
def check_order(order_id: str) -> str:
    """查询订单状态"""
    orders = {
        "ORD001": "已发货,预计 3 天送达",
        "ORD002": "处理中",
    }
    return orders.get(order_id, "未找到订单")

# 工具 2: 查询客服信息
@tool
def get_contact_info() -> str:
    """获取客服联系方式"""
    return "客服电话: 400-123-4567 | 邮箱: support@example.com"

# 创建 Agent
agent = create_agent(
    model="gpt-3.5-turbo",
    tools=[check_order, get_contact_info],
    system_prompt="你是一个智能客服,回答用户问题,必要时查询订单状态",
)

# 创建记忆
memory = ConversationBufferMemory()

# 用户: 查询订单
response1 = agent.invoke({
    "messages": [{"role": "user", "content": "我的订单 ORD001 在哪?"}]
}, memory=memory)
print(response1)

# 用户: 联系客服
response2 = agent.invoke({
    "messages": [{"role": "user", "content": "怎么联系客服?"}]
}, memory=memory)
print(response2)
```

## 💡 学习要点

### 1. Agent 的核心能力

| 能力 | 说明 |
|------|------|
| ✅ 理解用户意图 | Agent 知道用户想做什么 |
| ✅ 选择合适的工具 | Agent 决定需要调用什么工具 |
| ✅ 调用工具获取信息 | Agent 执行工具调用 |
| ✅ 综合信息回答 | Agent 整合结果给用户 |

### 2. Agent 的工作流程

```
用户问题
    ↓
Agent 理解意图
    ↓
选择工具
    ↓
调用工具
    ↓
获取结果
    ↓
综合回答
```
