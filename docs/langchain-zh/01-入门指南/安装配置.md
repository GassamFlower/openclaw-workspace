# 安装配置 LangChain

> 从零开始搭建 LangChain 开发环境

## 🛠️ 环境准备

### 1. 安装 Python

LangChain 基于 Python,建议使用 Python 3.8+:

```bash
# 检查 Python 版本
python --version
# 推荐: 3.10 或更高版本

# 如果没有安装,从官网下载: https://www.python.org/downloads/
```

### 2. 安装 VS Code

推荐使用 VS Code 作为编辑器:
- 下载: https://code.visualstudio.com/

## 📦 安装 LangChain

### 方式一: 使用 pip (推荐)

```bash
# 安装核心包
pip install langchain

# 安装常用模型集成 (根据需要选择)
pip install langchain-openai        # OpenAI GPT
pip install langchain-anthropic     # Anthropic Claude
pip install langchain-google-genai   # Google PaLM/Gemini
pip install langchain-community      # 社区工具和数据库
pip install langchain-experimental    # 实验性功能
pip install langchain-mcp            # MCP 支持

# 安装 LangSmith (调试工具)
pip install langsmith
```

### 方式二: 使用虚拟环境 (最佳实践)

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 升级 pip
pip install --upgrade pip

# 安装 LangChain
pip install langchain langchain-openai
```

## 🔑 配置 API 密钥

### OpenAI

```bash
# 设置环境变量
set OPENAI_API_KEY=your-api-key-here
```

或者在代码中设置:

```python
import os
os.environ["OPENAI_API_KEY"] = "sk-xxxxxxxxxxxxxxxxxxxx"
```

### Anthropic Claude

```bash
# 设置环境变量
set ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxxxx
```

### Google PaLM/Gemini

```bash
# 设置环境变量
set GOOGLE_API_KEY=your-google-api-key
```

## ✅ 验证安装

创建一个测试脚本 `test_install.py`:

```python
from langchain_openai import ChatOpenAI

# 初始化模型
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)

# 测试调用
response = llm.invoke("你好! 请用一句话介绍 LangChain。")
print(response.content)
```

运行测试:

```bash
python test_install.py
```

如果看到输出,说明安装成功!

## 🎯 快速体验

### 最简单的示例

```python
from langchain_openai import ChatOpenAI

# 创建模型
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)

# 对话
response = llm.invoke("用一句话解释什么是人工智能")
print(response.content)
```

### 使用 Agent

```python
from langchain.agents import create_agent

# 创建 Agent
agent = create_agent(
    model="gpt-4",
    tools=[],  # 先不提供工具
    system_prompt="你是一个有用的助手",
)

# 运行
result = agent.invoke({"messages": [{"role": "user", "content": "你好!"}]})
print(result)
```

## 🔧 环境变量管理

### 创建 `.env` 文件

在项目根目录创建 `.env` 文件:

```
OPENAI_API_KEY=sk-xxxxxxxx
ANTHROPIC_API_KEY=sk-ant-xxxxxxxx
GOOGLE_API_KEY=your-google-key
```

### 使用 python-dotenv

```bash
pip install python-dotenv
```

在代码中加载:

```python
from dotenv import load_dotenv
load_dotenv()  # 自动加载 .env 文件
```

## 🐛 常见问题

### 问题 1: pip 安装失败

```bash
# 升级 pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install langchain -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题 2: API 密钥无效

确保:
- API 密钥正确
- 已设置环境变量
- API 有余额

### 问题 3: 版本冲突

```bash
# 创建新的虚拟环境
python -m venv venv
source venv/bin/activate
pip install langchain langchain-openai
```
