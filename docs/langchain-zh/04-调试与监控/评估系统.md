## 💻 基础评估方法

### 1. 手动评估

```python
# 手动评估输出质量
result = llm.invoke("问题")

# 评估维度:
# - 准确性: 答案是否正确?
# - 相关性: 是否回答了问题?
# - 完整性: 是否包含所有要点?
# - 清晰度: 是否容易理解?

# 示例
questions = [
    "什么是 LangChain?",
    "如何安装 LangChain?",
    "LangChain 支持哪些模型?",
]

for question in questions:
    result = llm.invoke(question)
    print(f"问题: {question}")
    print(f"答案: {result.content}")
    print("---")
````

### 2. 自动化评估

```python
from langchain.evaluation import load_evaluator

# 加载评估器
evaluator = load_evaluator(
    "labeled_criteria",
    criteria={
        "accuracy": "答案是否准确?",
        "relevance": "答案是否相关?",
    }
)

# 评估
prediction = llm.invoke("问题")
result = evaluator.evaluate_strings(
    prediction=prediction.content,
    reference="标准答案"
)

print(result)
# {'criteria': {'accuracy': True, 'relevance': True}}
````

## 🛠️ LangSmith 评估

### 1. 使用 LangSmith 评估器

```python
from langsmith.evaluation import evaluate

# 评估数据集
dataset = [
    {"input": "什么是 LangChain?", "expected": "LangChain 是一个 AI 应用开发框架"},
    {"input": "如何安装?", "expected": "使用 pip install langchain"},
]

# 评估
results = evaluate(
    dataset,
    evaluator="llm",
    config={"name": "LangChain Evaluation"},
)

print(results)
````

### 2. 创建自定义评估器

```python
from langchain.evaluation import StringEvaluator

class MyEvaluator(StringEvaluator):
    def evaluate_string(self, prediction: str, reference: str) -> dict:
        # 自定义评估逻辑
        if prediction == reference:
            return {"score": 1, "reason": "完美匹配"}
        else:
            return {"score": 0, "reason": "不匹配"}

# 使用
evaluator = MyEvaluator()
result = evaluator.evaluate_string(
    prediction="答案",
    reference="标准答案"
)
`````

## 🎯 A/B 测试

### 1. 基础 A/B 测试

```python
# 对比不同模型
from langchain_openai import ChatOpenAI

# 模型 A
llm_a = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)
results_a = []

# 模型 B
llm_b = ChatOpenAI(model="gpt-4", temperature=0)
results_b = []

# 测试数据
test_questions = [
    "什么是 LangChain?",
    "如何安装 LangChain?",
    "LangChain 支持哪些模型?",
]

# 运行测试
for question in test_questions:
    result_a = llm_a.invoke(question).content
    result_b = llm_b.invoke(question).content

    results_a.append({"question": question, "answer": result_a})
    results_b.append({"question": question, "answer": result_b})

# 对比
print("GPT-3.5:", results_a)
print("\nGPT-4:", results_b)
````

### 2. 使用 LangSmith 进行 A/B 测试

```python
from langsmith import Client

client = Client()

# 创建实验
experiment_a = client.create_experiment(
    name="GPT-3.5",
    dataset_name="langchain-questions",
    parameters={"model": "gpt-3.5-turbo"},
)

experiment_b = client.create_experiment(
    name="GPT-4",
    dataset_name="langchain-questions",
    parameters={"model": "gpt-4"},
)

# 运行实验
results_a = client.run_experiment(experiment_a)
results_b = client.run_experiment(experiment_b)

# 对比结果
print(f"GPT-3.5 成本: ${results_a['cost']:.2f}")
print(f"GPT-4 成本: ${results_b['cost']:.2f}")
`````

## 💡 评估流程

### 完整的评估流程

```
1. 准备测试数据
   - 选择代表性问题
   - 准备标准答案

2. 运行评估
   - 对比不同模型
   - 记录所有输出

3. 分析结果
   - 统计指标
   - 识别问题

4. 优化改进
   - 调整 Prompt
   - 优化配置

5. 验证效果
   - 再次评估
   - 对比改进
``````

## 🚀 高级评估

### 1. A/B 测试框架

```python
class ABTestFramework:
    def __init__(self, models):
        self.models = models

    def run_test(self, test_data):
        results = {}

        for model_name, llm in self.models.items():
            results[model_name] = []

            for item in test_data:
                prediction = llm.invoke(item["question"]).content
                score = evaluate(prediction, item["expected"])
                results[model_name].append(score)

        return results

# 使用
framework = ABTestFramework({
    "gpt-3.5": ChatOpenAI(model="gpt-3.5-turbo"),
    "gpt-4": ChatOpenAI(model="gpt-4"),
})

results = framework.run_test(test_data)
print(results)
````
