### 1. 错误处理

```python
from flask import Flask
import logging

app = Flask(__name__)
logging.basicConfig(level=logging.ERROR)

@app.errorhandler(Exception)
def handle_error(e):
    logging.error(f"Error: {e}")
    return jsonify({"error": "Internal Server Error"}), 500
````
