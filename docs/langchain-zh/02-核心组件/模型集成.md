# 模型集成 - LLM 提供商使用指南

> LangChain 支持多种 LLM 提供商,提供统一的接口

## 🎯 什么是模型集成?

### 简单理解

**模型集成 = 使用不同的 AI 模型提供商**

LangChain 提供统一的接口,你可以:
- 用 OpenAI 的 GPT-4
- 用 Anthropic 的 Claude
- 用 Google 的 Gemini
- 用本地模型
- 甚至用自定义模型

**类比**:
- 就像你可以在手机上用不同的 App
- LangChain 提供统一的"启动键"
- 不需要关心每个 App 的具体使用方法

## 💻 常用 LLM 提供商

### 1. OpenAI

```python
from langchain_openai import ChatOpenAI

# GPT-4
llm = ChatOpenAI(
    model="gpt-4",
    temperature=0.7,
    max_tokens=1024,
    api_key="your-api-key"
)

# GPT-3.5
llm = ChatOpenAI(
    model="gpt-3.5-turbo",
    temperature=0.7
)

# 使用
response = llm.invoke("你好!")
print(response.content)
```

### 2. Anthropic Claude

```python
from langchain_anthropic import ChatAnthropic

# Claude 3 Sonnet
llm = ChatAnthropic(
    model="claude-sonnet-4-6",
    temperature=0.7,
    api_key="your-api-key"
)

# Claude 3 Opus
llm = ChatAnthropic(
    model="claude-opus-4",
    temperature=0.7
)

# 使用
response = llm.invoke("用中文介绍你自己")
```

### 3. Google PaLM/Gemini

```python
from langchain_google_genai import ChatGoogleGenerativeAI

# Gemini Pro
llm = ChatGoogleGenerativeAI(
    model="gemini-pro",
    temperature=0.7,
    google_api_key="your-api-key"
)

# Gemini Pro Vision
llm = ChatGoogleGenerativeAI(
    model="gemini-pro-vision",
    temperature=0.7
)
```

### 4. 本地模型 (Ollama)

```python
from langchain_community.chat_models import ChatOllama

# 使用 Ollama 运行的模型
llm = ChatOllama(
    model="llama2",
    temperature=0.7,
    base_url="http://localhost:11434"
)

# 使用
response = llm.invoke("你好!")
```

### 5. HuggingFace

```python
from langchain_community.llms import HuggingFaceHub

llm = HuggingFaceHub(
    repo_id="google/flan-t5-large",
    task="text-generation",
    huggingfacehub_api_token="your-token"

```

## 🎮 LLM 参数详解

### temperature

```python
# temperature 影响输出的随机性

temperature=0.0  # 确定性输出,适合事实性问题
temperature=0.3  # 稍微随机,适合创意写作
temperature=0.7  # 平衡,适合大多数场景
temperature=1.0  # 高度随机,适合头脑风暴

# 示例
llm = ChatOpenAI(model="gpt-4", temperature=0.0)
```

### max_tokens

```python
# 最大输出 token 数

llm = ChatOpenAI(
    model="gpt-4",
    max_tokens=100  # 最多输出 100 个 token
)

# 如果需要长文本,设置更大的值
llm = ChatOpenAI(
    model="gpt-4",
    max_tokens=2048
)
```

### model

```python
# 选择不同的模型

ChatOpenAI(model="gpt-4")
ChatOpenAI(model="gpt-3.5-turbo")
ChatAnthropic(model="claude-sonnet-4-6")
ChatGoogleGenerativeAI(model="gemini-pro")
```

### streaming

```python
# 启用流式输出

from langchain.callbacks.streaming_stdout import StreamingStdOutCallbackHandler

llm = ChatOpenAI(
    model="gpt-4",
    streaming=True,
    callbacks=[StreamingStdOutCallbackHandler()]
)
```

## 🛠️ 统一接口

### 所有 LLM 都有相同的使用方式

```python
# ✅ 统一的调用方式

from langchain_openai import ChatOpenAI
from langchain_anthropic import ChatAnthropic
from langchain_google_genai import ChatGoogleGenerativeAI

# 创建不同提供商的 LLM
openai_llm = ChatOpenAI(model="gpt-4")
claude_llm = ChatAnthropic(model="claude-sonnet-4-6")
gemini_llm = ChatGoogleGenerativeAI(model="gemini-pro")

# 统一调用
response1 = openai_llm.invoke("你好")
response2 = claude_llm.invoke("你好")
response3 = gemini_llm.invoke("你好")
```

## 💡 选择 LLM 的技巧

### 1. 根据任务选择

```python
# 复杂推理 → GPT-4 / Claude
llm = ChatOpenAI(model="gpt-4")

# 速度快,成本敏感 → GPT-3.5 / 本地模型
llm = ChatOpenAI(model="gpt-3.5-turbo")

# 视觉任务 → GPT-4V / Gemini Pro Vision
llm = ChatOpenAI(model="gpt-4o")

# 代码生成 → Claude
llm = ChatAnthropic(model="claude-sonnet-4-6")
```

### 2. 成本考虑

```python
# GPT-4: 最强,最贵
llm = ChatOpenAI(model="gpt-4")

# GPT-3.5: 性价比高
llm = ChatOpenAI(model="gpt-3.5-turbo")

# 本地模型: 免费,但性能较弱
llm = ChatOllama(model="llama2")
```

### 3. 隐私考虑

```python
# 数据敏感 → 本地模型
llm = ChatOllama(model="llama2")

# 数据不敏感 → 云端模型
llm = ChatOpenAI(model="gpt-4")
```

## 🚀 进阶用法

### 1. 多模型对比

```python
# 对比不同模型的效果

models = [
    ChatOpenAI(model="gpt-3.5-turbo", temperature=0),
    ChatOpenAI(model="gpt-4", temperature=0),
    ChatAnthropic(model="claude-sonnet-4-6", temperature=0),
]

for model in models:
    response = model.invoke("总结 LangChain 的特点")
    print(f"{model.__class__.__name__}: {response.content}")
```

### 2. 模型切换

```python
# 根据需求动态切换模型

def get_llm(task_type: str):
    if task_type == "complex":
        return ChatOpenAI(model="gpt-4")
    elif task_type == "simple":
        return ChatOpenAI(model="gpt-3.5-turbo")
    else:
        return ChatOpenAI(model="gpt-4")

llm = get_llm("complex")
```

### 3. 模型池

```python
# 创建模型池,轮换使用

from langchain_openai import ChatOpenAI

class ModelPool:
    def __init__(self):
        self.models = [
            ChatOpenAI(model="gpt-3.5-turbo"),
            ChatOpenAI(model="gpt-4"),
        ]
        self.index = 0

    def get_model(self):
        model = self.models[self.index]
        self.index = (self.index + 1) % len(self.models)
        return model

model_pool = ModelPool()
llm = model_pool.get_model()
```

## 🎯 最佳实践

### 1. 环境变量管理

```bash
# .env 文件
OPENAI_API_KEY=sk-xxx
ANTHROPIC_API_KEY=sk-ant-xxx
GOOGLE_API_KEY=your-google-key
```

```python
from dotenv import load_dotenv
import os

load_dotenv()

llm = ChatOpenAI()
```

### 2. 超时设置

```python
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="gpt-4",
    request_timeout=30  # 30 秒超时
)
```

### 3. 重试机制

```python
from langchain_openai import ChatOpenAI
from tenacity import retry, stop_after_attempt

@retry(stop=stop_after_attempt(3))
def call_llm_with_retry(llm, prompt):
    return llm.invoke(prompt)

response = call_llm_with_retry(llm, "你好")
```
