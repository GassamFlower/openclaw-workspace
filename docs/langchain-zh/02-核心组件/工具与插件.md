# 工具与插件 - Agent 的能力扩展

> Tool 是 Agent 能够调用外部功能的关键

## 🎯 什么是 Tool?

### 简单理解

**Tool = Agent 的"手和脚"**

Agent 本身只会说话,但通过 Tool,它可以:
- 搜索网页
- 计算数学表达式
- 查询数据库
- 调用 API
- 执行代码

**类比**:
- Agent = 大脑
- Tool = 手和脚
- Agent + Tool = 完整的人

## 🔧 自定义工具

### 方式 1: 使用装饰器

```python
from langchain.tools import tool

@tool
def search_database(query: str) -> str:
    """在数据库中搜索用户信息。

    Args:
        query: 搜索关键词

    Returns:
        搜索结果列表
    """
    # 实现你的搜索逻辑
    results = db.query(query)
    return str(results)

# 使用工具
tools = [search_database]
````

### 方式 2: 继承 BaseTool

```python
from langchain.tools import BaseTool
from pydantic import BaseModel

class CalculatorInput(BaseModel):
    expression: str

class CalculatorTool(BaseTool):
    name = "calculator"
    description = "计算数学表达式"
    args_schema = CalculatorInput

    def _run(self, expression: str):
        try:
            return str(eval(expression))
        except Exception as e:
            return f"计算错误: {e}"

# 使用工具
tools = [CalculatorTool()]
````

### 方式 3: 使用 LangChain Tools

```python
from langchain.tools import Tool

def multiply(a: int, b: int) -> int:
    """将两个数字相乘"""
    return a * b

tools = [
    Tool(
        name="Multiply",
        func=multiply,
        description="将两个数字相乘"
    )
]
```

## 🎮 Tool 使用示例

### 示例 1: 天气查询 Agent

```python
from langchain.agents import create_agent
from langchain.tools import tool
import requests

@tool
def get_weather(city: str) -> str:
    """查询指定城市的天气"""
    url = f"https://api.weather.com?city={city}"
    response = requests.get(url)
    return response.json()

# 创建 Agent
agent = create_agent(
    model="gpt-4",
    tools=[get_weather],
    system_prompt="你是一个天气助手",
)

# 运行
response = agent.invoke({"messages": [{"role": "user", "content": "北京天气如何?"}]})
```

### 示例 2: 数据分析 Agent

```python
from langchain.agents import create_agent
from langchain.tools import tool
import pandas as pd

@tool
def analyze_data(file_path: str) -> str:
    """分析 CSV 文件中的数据"""
    df = pd.read_csv(file_path)
    return f"数据行数: {len(df)}, 列数: {len(df.columns)}"

@tool
def plot_data(file_path: str) -> str:
    """生成数据图表"""
    df = pd.read_csv(file_path)
    df.plot()
    return "图表已生成"

# 创建 Agent
agent = create_agent(
    model="gpt-4",
    tools=[analyze_data, plot_data],
    system_prompt="你是一个数据分析师",
)
```

### 示例 3: 自定义业务工具

```python
@tool
def check_order_status(order_id: str) -> str:
    """查询订单状态

    Args:
        order_id: 订单号

    Returns:
        订单状态信息
    """
    orders = {
        "ORD001": "已发货,预计 3 天送达",
        "ORD002": "处理中",
    }
    return orders.get(order_id, "未找到订单")

@tool
def update_order_status(order_id: str, status: str) -> str:
    """更新订单状态

    Args:
        order_id: 订单号
        status: 新状态
    """
    orders["ORD001"] = status
    return f"订单 {order_id} 已更新为 {status}"

# 创建 Agent
agent = create_agent(
    model="gpt-4",
    tools=[check_order_status, update_order_status],
    system_prompt="你是一个客服助手",
)
```

### 2. API 集成

```python
from langchain.tools import Tool

def call_api(url: str, params: dict):
    """调用外部 API"""
    response = requests.get(url, params=params)
    return response.json()

tools = [
    Tool(
        name="GitHub Search",
        func=call_api,
        description="搜索 GitHub 仓库",
        args={"url": "https://api.github.com/search/repositories", "params": {}}
    )
]
```

### 3. 数据库工具

```python
    Tool(
        name="QueryDatabase",
        func=db.run,
        description="查询 SQL 数据库"
    )
]
```

## 💡 Tool 设计技巧

### 技巧 1: 清晰的描述

```python
# ❌ 不好的描述
@tool
def my_tool(data): ...

# ✅ 好的描述
@tool
def search_database(query: str) -> str:
    """在数据库中搜索用户信息。

    Args:
        query: 搜索关键词

    Returns:
        搜索结果列表

    使用场景:
        - 查询用户信息
        - 检查订单状态
        - 搜索产品
    """
    ...
```

### 技巧 2: 明确的参数

```python
@tool
def calculate(expression: str) -> str:
    """计算数学表达式。

    Args:
        expression: 要计算的数学表达式,例如 "1 + 2 * 3"

    Returns:
        计算结果
    """
    ...
```

### 技巧 3: 错误处理

```python
@tool
def safe_search(query: str) -> str:
    """安全搜索。

    Args:
        query: 搜索关键词

    Returns:
        搜索结果,如果出错返回错误信息
    """
    try:
        result = api.search(query)
        return result
    except Exception as e:
        return f"搜索失败: {str(e)}"
```

### 技巧 4: 参数验证

```python
from pydantic import BaseModel, validator

class CalculatorInput(BaseModel):
    a: float
    b: float

    @validator('b')
    def validate_b(cls, v):
        if v == 0:
            raise ValueError("除数不能为 0")
        return v

class CalculatorTool(BaseTool):
    args_schema = CalculatorInput

    def _run(self, a: float, b: float):
        return a / b
```

## 📚 工具最佳实践

### 1. 工具数量

```python
# ✅ 保持工具数量适中 (3-5 个)
tools = [search_tool, calculator, api_tool, db_tool]

# ❌ 不要给太多工具
tools = [tool1, tool2, tool3, tool4, tool5, tool6, ...]
# 问题: Agent 不知道用哪个
```

```python
# ✅ 描述清晰
@tool
def search_database(query: str) -> str:
    """搜索数据库,返回相关记录。用于查询用户、订单等数据。"""

def search(query): ...
```

## 🚀 进阶用法

### 1. 工具链

```python
# 先搜索,再处理
@tool
def search_then_process(query: str) -> str:
    """搜索并处理数据"""
    search_result = search_tool.run(query)
    processed = process_data(search_result)
    return processed

tools = [search_then_process]
```

### 2. 条件工具

```python
@tool
def conditional_tool(query: str) -> str:
    """根据查询类型选择工具"""
    if "天气" in query:
        return weather_tool.run(query)
    elif "计算" in query:
        return calculator.run(query)
    else:
        return search_tool.run(query)

tools = [conditional_tool]
```

### 3. 工具调用 hooks

```python
from langchain.tools import ToolCall

def before_tool_call(tool: ToolCall):
    print(f"即将调用工具: {tool.name}")
    print(f"参数: {tool.args}")

tool = Tool(
    func=my_function,
    callbacks=[before_tool_call]
)
```
