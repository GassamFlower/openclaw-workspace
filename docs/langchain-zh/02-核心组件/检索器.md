# 检索器 - RAG 应用核心

> Retriever 是 RAG (检索增强生成) 应用的核心组件

## 🎯 什么是检索器?

### 简单理解

**检索器 = 从知识库中查找相关信息**

RAG = Retrieval Augmented Generation = 检索增强生成

**工作流程**:
```
用户问题
    ↓
向量化
    ↓
在向量数据库中搜索
    ↓
找到相关文档
    ↓
把文档 + 问题一起喂给 LLM
    ↓
LLM 基于文档回答
````

**类比**:
- 用户 = 顾客
- LLM = 不知道具体知识,但会说话
- 检索器 = 图书管理员
- 知识库 = 书店

用户问问题 → 检索器去书店找书 → 把书拿给 LLM → LLM 基于书回答

## 💻 RAG 完整示例

### 最简单的 RAG

```python
from langchain_community.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings
from langchain.chains import RetrievalQA

# 1. 创建知识库
knowledge_base = [
    "LangChain 是一个 AI 应用开发框架",
    "LangChain 支持 OpenAI、Claude 等模型",
    "LangChain 提供工具调用和记忆功能",
]

# 2. 创建向量数据库
vectorstore = FAISS.from_texts(
    texts=knowledge_base,
    embedding=OpenAIEmbeddings(),
)

# 3. 创建检索器
retriever = vectorstore.as_retriever()

# 4. 创建 RAG Chain
qa_chain = RetrievalQA.from_chain_type(
    llm=ChatOpenAI(model="gpt-4"),
    chain_type="stuff",
    retriever=retriever,
)

# 5. 问答
response = qa_chain.invoke("LangChain 支持哪些模型?")
print(response["result"])
```

## 🛠️ 检索器类型

### 1. 向量检索器

```python
from langchain_community.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings

# 创建向量数据库
vectorstore = FAISS.from_texts(
    texts=["文档1", "文档2", "文档3"],
    embedding=OpenAIEmbeddings(),
)

# 创建检索器
retriever = vectorstore.as_retriever()
````

### 2. 存储检索器 (Memory Store)

```python
from langchain.storage import InMemoryStore

# 创建内存存储
store = InMemoryStore()

# 创建检索器
retriever = store.as_retriever(
    namespace="documents",
    search_kwargs={"k": 3}
)
````

### 3. 自定义检索器

```python
from langchain.retrievers import BaseRetriever
from pydantic import BaseModel

class CustomRetriever(BaseRetriever, BaseModel):
    search_kwargs: dict

    def _get_relevant_documents(self, query: str):
        # 自定义检索逻辑
        # 例如: 查询数据库、搜索网页等
        documents = self.search(query)
        return documents

# 使用
retriever = CustomRetriever(search_kwargs={"k": 5})
````

## 🔍 检索器参数

### k (返回文档数量)

```python
# 返回最相关的 3 个文档
retriever = vectorstore.as_retriever(search_kwargs={"k": 3})

# 返回最相关的 5 个文档
retriever = vectorstore.as_retriever(search_kwargs={"k": 5})
```

### score_threshold (相似度阈值)

```python
# 只返回相似度高于 0.7 的文档
retriever = vectorstore.as_retriever(
    search_kwargs={"k": 3, "score_threshold": 0.7}
)
```

### search_type (检索类型)

```python
# 相似度检索 (默认)
retriever = vectorstore.as_retriever(
    search_type="similarity"
)

# 相似度阈值
retriever = vectorstore.as_retriever(
    search_type="similarity_score_threshold",
    search_kwargs={"score_threshold": 0.7}
)

# 最大边际相关 (MMR)
retriever = vectorstore.as_retriever(
    search_type="mmr",
    search_kwargs={"k": 3, "fetch_k": 10}
)
```

## 📊 常用向量数据库

### 1. FAISS (Facebook AI Similarity Search)

```python
from langchain_community.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings

# 创建
vectorstore = FAISS.from_texts(
    texts=["文档1", "文档2"],
    embedding=OpenAIEmbeddings(),
)

# 保存
vectorstore.save_local("faiss_index")

# 加载
new_vectorstore = FAISS.load_local(
    "faiss_index",
    OpenAIEmbeddings(),
    allow_dangerous_deserialization=True
)

# 使用
retriever = new_vectorstore.as_retriever()
```

### 2. Chroma

```python
from langchain_community.vectorstores import Chroma
from langchain_openai import OpenAIEmbeddings

# 创建
vectorstore = Chroma.from_texts(
    texts=["文档1", "文档2"],
    embedding=OpenAIEmbeddings(),
    persist_directory="./chroma_db"
)

# 加载
vectorstore = Chroma(
    persist_directory="./chroma_db",
    embedding_function=OpenAIEmbeddings()
)

# 使用
retriever = vectorstore.as_retriever()
```

### 3. Pinecone

```python
from langchain_community.vectorstores import Pinecone

vectorstore = Pinecone.from_texts(
    texts=["文档1", "文档2"],
    embedding=OpenAIEmbeddings(),
    index_name="my_index"
)

retriever = vectorstore.as_retriever()
```

## 💡 高级检索技巧

### 技巧 1: 混合检索

```python
# 关键词检索 + 向量检索

from langchain.retrievers import EnsembleRetriever
from langchain_community.retrievers import BM25Retriever

# 向量检索器
vector_retriever = vectorstore.as_retriever()

# BM25 检索器 (关键词)
bm25_retriever = BM25Retriever.from_documents(documents)

# 合并检索器
ensemble_retriever = EnsembleRetriever(
    retrievers=[vector_retriever, bm25_retriever],
    weights=[0.7, 0.3]
)
```

### 技巧 2: 多路检索

```python
# 从多个知识库检索

from langchain.retrievers import MultiQueryRetriever

# 主检索器
retriever = vectorstore.as_retriever()

# 多路检索器
multi_retriever = MultiQueryRetriever.from_llm(
    retriever=retriever,
    llm=ChatOpenAI(model="gpt-3.5-turbo")
)
```

### 技巧 3: 自定义重排序

```python
# 先检索,再重排序

from langchain.retrievers import ContextualCompressionRetriever
from langchain.retrievers.document_compressors import LLMChainExtractor

# 检索器
retriever = vectorstore.as_retriever()

# 压缩器
compressor = LLMChainExtractor.from_llm(
    llm=ChatOpenAI(model="gpt-3.5-turbo")
)

# 压缩检索器
compression_retriever = ContextualCompressionRetriever(
    base_compressor=compressor,
    base_retriever=retriever
)
```

## 🎯 RAG 应用示例

### 示例 1: 问答系统

```python
from langchain.chains import RetrievalQA
from langchain_community.vectorstores import FAISS
from langchain_openai import OpenAIEmbeddings

# 创建知识库
knowledge_base = [
    "LangChain 是一个 AI 应用开发框架",
    "LangChain 支持多种 LLM 提供商",
    "LangChain 提供工具调用功能",
]

# 创建向量数据库
vectorstore = FAISS.from_texts(
    texts=knowledge_base,
    embedding=OpenAIEmbeddings(),
)

# 创建 RAG Chain
qa_chain = RetrievalQA.from_chain_type(
    llm=ChatOpenAI(model="gpt-4"),
    chain_type="stuff",
    retriever=vectorstore.as_retriever(),
)

# 问答
question = "LangChain 支持哪些功能?"
answer = qa_chain.invoke(question)["result"]
print(answer)
```

### 示例 2: 文档问答

```python
from langchain_community.vectorstores import Chroma
from langchain_openai import ChatOpenAI, OpenAIEmbeddings
from langchain.chains import RetrievalQA

# 从文档创建知识库
documents = [
    "LangChain 是用于开发 LLM 应用的框架",
    "LangChain 支持 Chain、Agent 等核心组件",
    "LangChain 可以与各种工具集成",
]

vectorstore = Chroma.from_documents(
    documents=documents,
    embedding=OpenAIEmbeddings(),
)

# 创建问答链
qa_chain = RetrievalQA.from_chain_type(
    llm=ChatOpenAI(model="gpt-4"),
    chain_type="stuff",
    retriever=vectorstore.as_retriever(),
    return_source_documents=True,
)

# 问答
result = qa_chain.invoke("什么是 LangChain?")
print(result["result"])

# 查看参考文档
for doc in result["source_documents"]:
    print(doc.page_content)
```

## 📊 性能优化

### 1. 选择合适的 k 值

```python
# k 太小: 信息不足
retriever = vectorstore.as_retriever(k=1)

# k 太大: 信息过多,影响性能
retriever = vectorstore.as_retriever(k=20)

# 适中: 3-5
retriever = vectorstore.as_retriever(k=3)
```

### 2. 使用 MMR 检索

```python
# MMR 确保检索结果的多样性
retriever = vectorstore.as_retriever(
    search_type="mmr",
    search_kwargs={"k": 3, "fetch_k": 10}
)
```

### 3. 缓存结果

```python
from langchain.cache import InMemoryCache

# 启用缓存
from langchain.globals import set_llm_cache
set_llm_cache(InMemoryCache())

# 相同的问题不会重复调用 LLM
```
