# 09 - 监控分析

> 监控 Dify 应用性能和使用情况

## 📊 监控功能

| 功能 | 说明 |
|------|------|
| **性能监控** | 响应时间、成功率 |
| **用户分析** | 使用统计、用户行为 |
| **日志管理** | 实时日志、历史记录 |
| **反馈收集** | 用户评价、建议 |

---

## 📈 性能监控

### 查看性能数据

1. 左侧菜单 → "监控"
2. 查看各项指标

### 关键指标

- **响应时间**：平均响应时间
- **成功率**：请求成功率
- **Token 使用量**：模型调用次数

---

## 📝 日志管理

### 实时日志

1. 左侧菜单 → "调试"
2. 查看实时日志

### 日志内容

```
[INFO] [2026-04-01 10:00:00] Node: llm-1 | Input: "你好"
[INFO] [2026-04-01 10:00:01] Node: llm-1 | Output: "你好！有什么可以帮你的吗？"
[INFO] [2026-04-01 10:00:02] Workflow completed | Status: success
```
