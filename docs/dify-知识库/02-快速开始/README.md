# 02 - 快速开始

> 30 分钟上手 Dify，创建你的第一个 AI 应用

## 🎯 目标

在 30 分钟内创建一个简单的聊天机器人，并能回答问题

## 📋 前置要求

- 浏览器（推荐 Chrome、Edge）
- API Key（从 OpenAI、Claude 等获取）
- 30 分钟时间

---

## 步骤 1：访问 Dify

### 方式 A：Dify Cloud（推荐新手）

1. 访问：https://cloud.dify.ai
2. 注册账号（免费）

### 方式 B：自托管（完全免费）

```bash
# 使用 Docker 快速部署
docker run -d --name dify -p 80:3000 \
  -v ./data:/app/data \
  langgenius/dify-api:latest

# 访问 http://localhost
```

---

## 步骤 2：创建应用

1. 登录 Dify
2. 点击 "创建应用"
3. 选择 "聊天机器人（Chatbot）"
4. 输入应用名称（如 "我的第一个机器人"）
5. 点击 "创建"

---

## 步骤 3：配置 LLM

1. 在左侧菜单找到 "模型提供商"
2. 添加你的模型（以 OpenAI 为例）
3. 粘贴 API Key
4. 选择模型（如 `gpt-4o` 或 `gpt-3.5-turbo`）
5. 保存

> **💡 小提示**：如果没有 API Key，可以使用本地模型（详见 [本地模型](../07-API开发/LLM-配置.md#本地模型)）

---

## 步骤 4：设计工作流

1. 进入应用的 "工作流" 页面
2. 你会看到一个简单的流程：`用户输入 → LLM → 回复`

### 可视化编辑

- **拖拽节点**：从左侧工具栏拖到画布
- **连接节点**：点击节点间的线条进行连接
- **配置节点**：双击节点进行配置

---

## 步骤 5：测试应用

1. 在页面顶部点击 "试运行"（或 "测试"）
2. 输入问题："你好"
3. 查看回复

如果看到 AI 回复了，恭喜！🎉

---

## 步骤 6：发布应用

### 方式 A：Web App

1. 点击 "发布"
2. 选择 "Web App"
3. 输入自定义域名（可选）
4. 点击 "发布"

### 方式 B：API

1. 点击 "发布"
2. 选择 "API"
3. 复制 API 端点

---

## 🎉 成功！

现在你的第一个 AI 应用已经上线了！

### 下一步

- 添加知识库功能
- 优化回复质量
- 集成到你的网站

---

## 📱 快速体验知识库

### 步骤 1：创建知识库

1. 点击 "知识库"
2. 点击 "创建知识库"
3. 输入名称（如 "公司文档"）
4. 点击 "创建"

### 步骤 2：上传文档

1. 选择文档类型（TXT、PDF、DOCX 等）
2. 上传文件
3. 等待处理完成

### 步骤 3：关联应用

1. 在应用工作流中添加 "知识库检索" 节点
2. 关联刚才创建的知识库
3. 测试提问

---

## 常见问题

### Q: 为什么 AI 回复不相关？

**A**: 可能的原因：
- 没有知识库，只依赖模型本身
- LLM 配置不合适
- 回复质量看模型选择

### Q: 能用本地模型吗？

**A**: 可以！在模型提供商配置中添加本地模型（如 Ollama、vLLM）

### Q: API Key 安全吗？

**A**: 如果自托管，API Key 保存在你自己的服务器上，更安全。Cloud 版本由 Dify 官方管理。
