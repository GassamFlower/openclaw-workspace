# 语法编程 - 语法编程

> 使用 SPSS 语法进行自动化分析

## 🎯 语法优势

- 可重复分析
- 批量处理
- 复杂操作简化

---

## 💻 基础语法示例

```
* 定义变量
DATA LIST / id (1-10) age gender (1,2).

* 定义变量标签
VARIABLE LABELS age '年龄' gender '性别'.

* 基础统计分析
DESCRIPTIVES VARIABLES=age /STATISTICS=MEAN STDDEV.

* 假设检验
T-TEST GROUPS=gender(1 2) /VARIABLES=age.
```
