# SPSS 快速开始

> 10 分钟内学会 SPSS 基本操作

## 🚀 第一步: 数据录入

### 创建数据文件

```python
# SPSS 数据录入步骤:

1. 打开 SPSS
2. 点击"变量视图"标签
3. 录入变量信息:
   - 名称: ID, 年龄, 性别, 工资
   - 类型: 数值, 数值, 字符, 数值
   - 标签: 编号, 年龄, 性别, 工资
4. 点击"数据视图"标签
5. 录入数据:
   ID  年龄  性别  工资
   1   25    男    5000
   2   30    女    6000
   3   28    男    5500
```

---

## 🎯 第二步: 描述性统计分析

### 计算基本统计量

```python
# 操作步骤:

1. 菜单: 分析 → 描述统计 → 描述
2. 选择变量: 年龄, 工资
3. 勾选统计量: 均值, 标准差, 最小值, 最大值
4. 点击"确定"

# 输出结果:
描述统计量

年龄
─────────────────────────────────────
N          3
均值        27.67
标准差      2.52
最小值      25
最大值      30

工资
─────────────────────────────────────
N          3
均值        5500
标准差      577.35
最小值      5000
最大值      6000
```

## 🔥 第三步: T检验

### 比较两组差异

```python
# 场景: 比较男女工资差异

# 操作步骤:

1. 菜单: 分析 → 假设检验 → 均值比较 → 独立样本 T检验
2. 变量组: 工资
3. 分组变量: 性别 (定义组: 男=1, 女=2)

# 输出结果:
独立样本检验

─────────────────────────────────────

年龄                    0.000  1.000

t 检验
─────────────────────────────────────
                    t      df   Sig. (双侧) 均值差值
工资                0.866  1    0.538     1000
```

## 💡 第五步: 数据清洗

### 处理缺失值

```python
# 场景: 检查缺失值并删除

# 操作步骤:

1. 菜单: 数据 → 选择个案
2. 选择"如果条件满足"

4. 点击"如果" → 输入条件 → 继续
5. 选择"过滤未选定的个案"
6. 点击"确定"

# 结果: 删除了年龄小于18的记录
```

## 🎮 实战示例: 员工数据分析

```python
# 完整案例: 分析员工数据

# 1. 数据录入
变量: ID, 姓名, 年龄, 性别, 部门, 工资
数据:
1, 张三, 25, 男, 研发, 5000
2, 李四, 30, 女, 市场, 6000
3, 王五, 28, 男, 销售, 5500
4, 赵六, 35, 男, 研发, 8000

# 2. 描述性统计
分析 → 描述统计 → 描述
结果:
- 平均年龄: 29.5岁
- 平均工资: 6125元
- 性别分布: 男3人, 女1人

# 3. T检验
分析 → 假设检验 → 均值比较 → 独立样本 T检验
结果:
- 男女工资无显著差异 (P > 0.05)

# 4. ANOVA
分析 → 一般线性模型 → 单因素方差分析
结果:
- 不同部门工资有显著差异 (P < 0.05)
```

## 📝 SPSS 常用菜单

### 分析菜单

```
分析 →
├─ 描述统计: 描述, 频率, 探索
├─ 假设检验: 均值比较, 卡方检验
├─ 一般线性模型: 方差分析
├─ 相关: 双变量, 偏相关
├─ 回归: 线性, 逻辑回归
└─ 非参数检验: 卡方, K-S 检验
```

### 图形菜单

```
图形 →
├─ 旧对话框:
│  ├─ 条形图: 简单, 群组, 堆积
│  ├─ 折线图: 简单, 多线, 区间
│  ├─ 散点图: 简单, 矩阵
│  └─ 饼图: 简单
└─ 图表构建器: 可视化创建
```

### 数据菜单

```
数据 →
├─ 数据排序: 排序个案
├─ 转置: 数据转置
├─ 选择个案: 筛选数据
├─ 拆分文件: 分组分析
└─ 合并文件: 数据合并
```

## 🎓 进阶技巧

### 1. 使用 SPSS 命令

```python
# 在语法编辑器中运行
DESCRIPTIVES VARIABLES=年龄 工资
  /STATISTICS=MEAN STDDEV MIN MAX.

FREQUENCIES VARIABLES=性别
  /ORDER=ANALYSIS.
```

### 2. 自定义变量标签

```python
# 在变量视图设置
```
