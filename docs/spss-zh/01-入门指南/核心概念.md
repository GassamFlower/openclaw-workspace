# SPSS 核心概念

> 掌握这些核心概念,你就理解了 SPSS 的工作原理

## 🎯 必知必会概念

### 1. 数据视图 (Data View)

**数据视图 = 看数据的地方**

```
┌────────┬───────┬───────┬────────┐
│ ID     │ 姓名  │ 年龄  │ 性别   │
├────────┼───────┼───────┼────────┤
│ 1      │ 张三  │ 25    │ 男     │
│ 2      │ 李四  │ 30    │ 女     │
│ 3      │ 王五  │ 28    │ 男     │
└────────┴───────┴───────┴────────┘
```

**特点**:
- 每一行 = 一个记录
- 每一列 = 一个变量
- 网格 = 数据单元格

### 2. 变量视图 (Variable View)

**变量视图 = 定义变量的地方**

```
┌──────────┬─────────┬────────┬───────┐
│ 名称     │ 标签    │ 类型   │ 宽度  │
├──────────┼─────────┼────────┼───────┤
│ ID       │ 编号    │ 数值   │ 8     │
│ 姓名     │ 姓名    │ 字符   │ 10    │
│ 年龄     │ 年龄    │ 数值   │ 3     │
│ 性别     │ 性别    │ 字符   │ 1     │
└──────────┴─────────┴────────┴───────┘
```

**变量属性**:
- **名称**: 变量名 (不能有空格)
- **标签**: 变量说明
- **类型**: 数值、字符、日期
- **宽度**: 数字位数
- **小数**: 小数位数

### 3. 变量 (Variable)

**变量 = 数据的列**

```python
# 示例: 员工数据

# 数值变量
- ID: 1, 2, 3, ...
- 年龄: 25, 30, 28, ...
- 工资: 5000, 8000, 6000, ...

# 字符变量
- 姓名: 张三, 李四, 王五
- 部门: 研发部, 市场部
- 地区: 北京, 上海
```

### 4. 观测值 (Case)

**观测值 = 数据的行**

```python
# 示例: 每行是一个观测值

# 观测值 1: ID=1, 姓名="张三", 年龄=25, 性别="男"
# 观测值 2: ID=2, 姓名="李四", 年龄=30, 性别="女"
# 观测值 3: ID=3, 姓名="王五", 年龄=28, 性别="男"
```

### 5. 缺失值 (Missing Value)

**缺失值 = 没有数据的单元格**

```python
# 示例

┌────────┬───────┬───────┬────────┐
│ ID     │ 年龄  │ 工资  │ 性别   │
├────────┼───────┼───────┼────────┤
│ 1      │ 25    │ 5000  │ 男     │
│ 2      │ 30    │ .     │ 女     │ ← 缺失值
│ 3      │ .     │ 6000  │ 男     │ ← 缺失值
└────────┴───────┴───────┴────────┘
```

### 6. 均值 (Mean)

**均值 = 平均值**

```python
# 计算年龄的平均值
data = [25, 30, 28]
mean = (25 + 30 + 28) / 3 = 27.67
```

### 7. 标准差 (Standard Deviation)

**标准差 = 数据的离散程度**

```python
# 标准差小 = 数据集中在平均值附近
# 标准差大 = 数据分布分散

# 示例
data1 = [27, 27, 27]  # 标准差 = 0
data2 = [10, 30, 44]  # 标准差 = 16.33
```

### 8. P值 (P-value)

**P值 = 统计显著性**

```python
# P值 < 0.05 = 显著
# P值 > 0.05 = 不显著

# 示例
t检验结果:
P值 = 0.03 < 0.05
结论: 差异显著
```

### 9. 显著性 (Significance)

**显著性 = 差异是否可信**

```python
# α = 0.05 (显著性水平)
# P < α → 差异显著
# P ≥ α → 差异不显著
```

### 10. 假设检验 (Hypothesis Testing)

**假设检验 = 统计推理**

```python
# 步骤:

# 原假设 (H0): 两组没有差异
# 备择假设 (H1): 两组有差异

# 收集数据 → 计算 P值 → 做决策

# 如果 P < 0.05 → 拒绝 H0
# 如果 P ≥ 0.05 → 不拒绝 H0
```

## 📊 常用统计方法

### 1. 描述性统计

```python
# 描述数据的整体特征

# 常用统计量:
- 均值 (Mean): 平均水平
- 中位数 (Median): 中间值
- 标准差 (Std Dev): 离散程度
- 偏度 (Skewness): 分布不对称
- 峰度 (Kurtosis): 分布尖锐程度

# SPSS 操作:
分析 → 描述统计 → 描述
```

### 2. T检验 (T-Test)

```python
# 比较两组的平均值差异

# 适用场景:
- 实验组 vs 对照组
- 男 vs 女
- 新产品 vs 旧产品

# 操作:
分析 → 假设检验 → 均值比较 → 独立样本 T检验
```

### 3. 方差分析 (ANOVA)

```python
# 比较多组的差异

# 适用场景:
- 三组以上对比
- 多个教学方法对比
- 多个药物效果对比

# 操作:
分析 → 一般线性模型 → 单因素方差分析
```

### 4. 相关分析 (Correlation)

```python
# 分析变量之间的关系

# 相关系数:
- r = 0: 无相关
- r = 0.3: 弱相关
- r = 0.6: 中等相关
- r = 0.8: 强相关

# 操作:
分析 → 相关 → 双变量
```

### 5. 回归分析 (Regression)

```python
# 建立变量间的预测模型

# 应用场景:
- 预测房价
- 预测销量
- 预测客户流失

# 操作:
分析 → 回归 → 线性
```
