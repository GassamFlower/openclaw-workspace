## 📋 文件说明

### docs/
所有文档，包括配置指南、使用说明等。

### memory/
每日工作总结，按日期保存。

### projects/
个人项目代码。

### scripts/
自动化脚本。

### templates/
常用模板，如计划模板、总结模板。

### archives/
旧项目或废弃内容。
