# 每日习惯养成方案

## 目标

养成以下好习惯：
1. 每天制定计划
2. 每天写总结
3. 定期休息（放下手机）

## 配置的提醒任务

### 1. 每日计划检查（10:00）

**触发时间**：每天 10:00
**飞书机器人提醒**：
```
🔥 罗永贵的每日计划检查提醒

时间：每天10:00

请回答以下问题：
1. 今天早上制定了计划吗？
2. 计划有哪些内容？
3. 今天有按时执行吗？

完成计划检查后，可以在飞书回复我。
```

### 2. 刷手机提醒（9:30-17:30）

**触发时间**：工作时段每2小时
**飞书机器人提醒**：
```
📱 刷手机提醒

时间：工作时段每2小时一次

你已经工作X小时了，该放下手机，起身活动一下。

建议：
- 做几个深蹲
- 看看窗外
- 喝口水
- 伸个懒腰

健康第一，效率第二。休息是为了走更远的路。
```

### 3. 每日总结（17:00）

**触发时间**：每天 17:00
**飞书机器人提醒**：
```
📝 每日总结提醒

时间：每天17:00

请完成今日总结：
1. 今天完成了哪些任务？
2. 有学到什么新东西？
3. 写了一篇文章或方案吗？
4. 明天计划做什么？

把你的总结写在 MEMORY.md 里，格式如下：
```markdown
## YYYY-MM-DD 每日总结

### 完成任务
- [ ] 任务1
- [ ] 任务2

### 学习收获
- 学到了什么

### 文章方案
- 写了什么

### 明日计划
- 计划做什么
```
```

## 执行流程

### 第1周
- [ ] 每天 10:00 回复计划检查
- [ ] 每 2 小时 放下手机
- [ ] 每天 17:00 回复总结

### 第2周
- [ ] 总结计划检查的规律
- [ ] 看看自己是否在拖延
- [ ] 根据实际情况调整计划

### 第3周
- [ ] 总结每日总结的模板
- [ ] 优化内容，更实用
- [ ] 形成自己的工作习惯
